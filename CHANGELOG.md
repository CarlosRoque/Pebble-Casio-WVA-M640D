# Changelog

## v1.0.1
### Improvements
- Day, date, Bluetooth, and DST indicators updated to dark grey for better contrast against the watch face
- Weather icon and temperature updated to white for improved readability

### Bug Fixes
- Fixed weather not loading on startup due to a JS crash in the settings handler
- Fixed battery display defaulting to percentage instead of the icon on first launch
- Fixed temperature unit setting not applying correctly before settings are saved

## v1.0.0 — Initial Release
- **Time & Date:** Full day-of-the-week name and day-of-the-month display
- **Live Weather:** Current conditions with a dedicated weather icon
- **System Status:** At-a-glance icons for Bluetooth connectivity and Daylight Saving Time (DST)
- **Battery Meter:** Integrated level indicator to keep you powered up
